Webtography

Clone a website via Wget

Webtography is sponsored by LoremCMS

Webtography uses Wget to mirror a website and pushes it to a repository.

  1. Create a GitHub token in settings. Only the public_repo scope is needed.
  2. Fill in the form below and soon after you will see the website mirrored in a new repository on your GitHub account.
